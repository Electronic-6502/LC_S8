# What is this ?
LC_S8 is a Open Source LC meter
can Measure Frequency, Capacitance (and ESR) and Inductance

# Specifications
 ## Power Supply
 - Voltage : 9-12V (9V Recommended)
 - Current : 150mA Max

 ## Frequency Meter :
 - Frequency Range : 2Hz - 8MHz
 - Resolution : 1Hz for F<10kHz and 0.01kHz for F>=10kHz
 - Tolerence : 0.1% Max
 - minimum Signal Amplitude : 0.2Vrms for F>50Hz and 1Vrms for F<=50Hz
 - Maximum Amplitude : 50Vpp

 ## Capacitance Meter :
 - Capacitance Range : 5pF to 50mF
 - Tolerence : 5% Max
 - ESR Range : 0.01Ω to 100Ω
 - Measure Stand-Alone Resistor : YES
 - Injected Input Voltage Range : -6V to +6V

 ## Inductance Meter 
 - Inductance Range : 0.01uH to 999H
 - Tolerence : 5% Max (NOT Guaranted !)
